dawidr.pl
Moja strona wizytowa, własnie ta na której właśnie się znajdujesz :) Służy głównie do prezentowania informacji o mojej osobie oraz obsługi mikro-bloga. Angular po stronie front-endu hostowany na firebase, back-end w django[rest] + MySQL na pythonanywhere.com.
Koreline
Aplikacja internetowa wspomagająca korepetycje online. Posiada system ogłoszeń z korepetycjami, panel użytkownika, panel nauczyciela, system płatności (symulowany), system wiadomości i powiadomień oraz system prowadzenia konwersacji audiowizualnej w czasie rzeczywistym. Aplikacja składa się z trzech części: aplikacja po stronie klienta zaimplementowana w Angularze 2 i Angular Material 2, restowe API w django[rest] + postgresql, serwer sygnalizujący WebRTC w nodejs przy użycie EasyRTC.
RPM-Solver
Program w Pythonie + numpy służący rozwiązywaniu test matryc ravena. Większość obliczeń wykonywane współbieżnie.
MusicStore
Restowe API do obsługi sklepu internetowego z płytami muzycznymi napisane przy użyciu django[rest], baza danych PostgreSQL.
Postman tour
Program rozwiązujący problem chińskiego listonosza w sposób rozproszony przy użyciu bibilioteki OpenMPI.
Webscraper zakładów bukmacherskich
Webscraper zbierający dane o zakładach bukmacherskich do bazy danych MongoDB. Program wieloprocesowy, w programie uruchominych jest 5 równolegle działających workerów które zbierają dane o meczach dla różnych przedziałów czasowych do zakończenia meczu (im bliżej wyniku tym częściej dane są zbierane). Do renderowania javascryptu na scrapowanych stronach wykorzystany został selenium webdriver.
niskiwklad.pl
Strona firmy prawniczej. Moim zadaniem była implementacja back-endu oraz wdrożenie na serwer produkcyjny. Aplikacja umożliwia edycje całej zawartości z poziomu panelu administratora oraz dodawanie aktualności.
OTK
Projekt wyszukiwarki telefonów komórkowych opartej na Semantic Web. Generowanie ontologii dla telefonów komórkowych w języku OWL przy użyciu biblioteki owlready. Dane pobierane z mgsm.pl za pomocą webscrapera. Przetwarzanie danych za pomocą języka SPARQL. Do obsługi wyszukiwarki aplikacja serwerowa we Flasku, po stronie klienta aplikacja w Angularze 2.
Rzemieślnicy
Aplikacja internetowa umożliwiająca wyszukiwanie lokalnych firm oferujących usługi (np. fryzjer, mechanik samochodowy). Integracja z OpenStreetMaps. Całość w Django + PostgreSQL i Materialize.
Resume problem
Program w Pythonie + gensim[word2vec] służący klasyfikacji danych zawartych w CV. Wykorzystanie wytrenowanego modelu od googla. Wejście: plik txt z CV; wyjście: xml ze sklasyfikowanymi danymi.
Dorabiaj
Aplikacja internetowa obsługująca ogłoszenia zlecenia drobnych prac (np. skoszenie trawnika, pomalowanie płotu). Klient w Angularze 2, back-end we Flasku + MongoDB.
Trenuj to
